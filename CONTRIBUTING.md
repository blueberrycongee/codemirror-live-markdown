# Contributing to codemirror-live-markdown

Thanks for your interest in contributing! Here's how to get started.

## Setup

```bash
git clone https://github.com/blueberrycongee/codemirror-live-markdown.git
cd codemirror-live-markdown
npm install
```

## Development

```bash
# Start dev server with hot reload
npm run dev

# Run the demo
cd demo && npx serve .
```

## Lint

```bash
npm run lint
npm run typecheck
```

## Test

```bash
npm test
```

## Pull Request Guidelines

1. Fork the repo and create your branch from `main`
2. Make sure tests pass (`npm test`)
3. Make sure linting passes (`npm run lint && npm run typecheck`)
4. Update documentation if needed
5. Submit your PR!
